## How to run project ?


1. Just run the **polishvodka.py**
2. It should be opened on 
````
http://localhost:5000/ 
````

OR 

Export your main python file as FLASK_APP
````
export FLASK_APP=polishvodka
````
then type and enter
````
flask run
````
